Initialize Timing Dialog

Initialize Timing Dialog koristi se za postavljanje inicijalnih vremenskih parametara nekog testbencha. Ovaj prozor otvara se samo jednom, prilikom stvaranja novog testbencha. Parametre poput Check Ouputs, Assign Inputs, Clock Time High, Clock Time Low, te Input Setup Time moguće je unijeti samo na ovom mjestu.

VHDL Lab pruža mogućnost stvaranja dva tipa testbencha: Single Clock Testbench i Combinatorial Testbench.

Single Clock Testbench

Ovaj tip testbencha može se stvoriti jedino ako model koji se testira sadrži barem jedan skalarni signal.

Upute za stvaranje novog Single Clock testbencha:

  1. U okviru Clock Information odabere se Single Clock, te ime signala koji će predstavljati clock.
  2. U Clock Timing Information okviru odabere se okidajući brid, padajući (Falling Edge) ili rastući (Rising Edge).
  3. U istom tom okviru podese se sljedeći parametri: Clock Time High, Clock Time Low, te Input Setup Time.
  4. U okviru Time Scale Information unese se duljina trajanja testbencha, te vremenska skala.

Combinatorial Testbench

Upute za stvaranje novog Combinatorial testbencha:

  1. U okviru Clock Information odabere se Combinatorial.
  2. U Combinatorial Timing Information okviru podese se Check Ouputs i Assign Inputs parametri.
  3. U okviru Time Scale Information unese se duljina trajanja testbencha, te vremenska skala.
