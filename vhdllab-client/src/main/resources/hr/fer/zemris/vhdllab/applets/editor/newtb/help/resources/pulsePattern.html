Pulse Pattern

Parametar Značenje
Initial value Početna vrijednost signala.
Initial delay Duljina za vrijeme koje signal ima inicijalnu vrijednost, mjeri se u periodima.
Pulse value Vrijednost pulsa. Mora biti suprotna od početne vrijednosti (Initial value).
Pulse length Duljina za vrijeme koje signal ima vrijednost pulsa, mjeri se u periodima.
